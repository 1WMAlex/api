API Documentation - GunGens Bank System

Welcome to the API documentation for The Guardian Bank System. Below you will find the available endpoints, their purposes, request formats, and response formats.

List All Users

Endpoint

GET /v1/users

Description

Retrieves a list of all users.

Request

GET /v1/users
Header: api-key: your_api_key

Response

{
    "users": [
        {
            "username": "john_doe",
            "balance": 1000,
            "banking_id": "jdoe123"
        },
        ...
    ]
}

Get User Information

Endpoint

GET /v1/user/:player_name

Description

Retrieves information about a specific user.

Request

GET /v1/user/:player_name
Header: api-key: your_api_key

Response

{
    "username": "john_doe",
    "balance": 1000,
    "banking_id": "jdoe123"
}

Get User Balance

Endpoint

GET /v1/user/:player_name/balance

Description

Retrieves the balance of a specific user.

Request

GET /v1/user/:player_name/balance
Header: api-key: your_api_key

Response

{
    "balance": 1000
}

Get User Bank Username

Endpoint

GET /v1/user/:player_name/bank_username

Description

Retrieves the bank username of a specific user.

Request

GET /v1/user/:player_name/bank_username
Header: api-key: your_api_key

Response

{
    "banking_id": "jdoe123"
}
